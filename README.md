# Робот "Анатолий"

## Авторство
Проект разработан:
- Тчанников Алексей
Дата начала разработки: 2025 г.
## Описание
Автономный робот для сбора и сортировки мусора в коттеджных поселках Московской области.
## Назначение
Решение проблемы утилизации отходов и улучшение экологической ситуации.
## Состав проекта
- Презентация
- Техническое описание
- Прототип робота
- Программная часть
## Примечание
Данный репозиторий создан для фиксации авторства и этапов разработки проекта.
